Module 3: Internet Security

3.4.1 Security-Related Indicators
  1. Unauthorized Access Attempts:
    Unexpected logins or access attempts to sensitive criminal justice databases or systems could indicate unauthorized access or potential security breaches.


  2. Suspicious Network Activity:
    Anomalies in network traffic, such as unusual data transfers or connections to suspicious IP addresses, may signal attempted cyber-attacks or data exfiltration.

  3. Unauthorized File Access:
    Instances of unauthorized access to confidential case files or evidence repositories may indicate insider threats or compromised user accounts.

  4. Phishing Attempts:
    Emails or messages requesting sensitive information, such as login credentials or personal details, could be phishing attempts targeting criminal justice personnel.


  5. Malware Infections:
    Signs of malware activity, such as unexpected system slowdowns, file modifications, or antivirus alerts, may indicate compromised devices within the criminal justice network.

  6. Security Configuration Changes:
    Unexpected alterations to security settings or access controls could suggest unauthorized modifications aimed at exploiting vulnerabilities or bypassing security measures.

  7. Security Alerts and Incident Reports:
    Notifications from security tools or incident reports highlighting unusual activities or potential security incidents should be promptly investigated to mitigate threats and protect sensitive data.

  8. Compliance Violations:
    Instances of non-compliance with security policies, data protection regulations, or criminal justice standards should be identified and addressed to prevent security breaches and legal liabilities.
