Module 2: Computer Security

2.2.1 Security Updates

Regularly Updating software and security is a crucial to maintaining a proper cybersecurity posture. It requires the systematic updating of software on computers to protect against vulnerabilities.

  • Automatic Updates:
    Both Windows and macOS operating systems feature built-in mechanisms that regularly check for software updates over the internet. These services ensure that your computer system is always equipped with the latest security features and fixes.
  • Update Services:
    For a computer to receive these updates, it must be configured to interact with the vendor's update service. This ensures that any patches released by the vendor are automatically applied to your system.
  • Security Patches:
    Often referred to simply as patches, these updates target specific vulnerabilities within the system. By addressing these vulnerabilities, patches play a critical role in fortifying the system against potential exploits and malware infections.
  • Instructional Resources:
  • For Windows users, updates can be managed and reviewed at the Windows Update page
  • For MacOS users, guidance on updates is available at th MacOS Update Guide
